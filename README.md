Proyecto PNFI en Informática

Este proyecto es una base de datos que busca resolver los problemas en procesos manuales de inventario y perdida de este mismo

Problemas a Resolver
1. Gestión Manual Ineficiente
El principal desafío es eliminar el control manual de los procesos de la biblioteca, como el registro de préstamos, devoluciones e inventario. Este enfoque genera errores, pérdida de datos y lentitud en la atención al usuario.

2. Dificultad en la Localización de Información
La falta de un sistema automatizado dificulta la búsqueda de la disponibilidad de un libro o el historial de un usuario, convirtiendo este proceso en algo tardío y complejo.

Objetivos del Proyecto
1. Automatizar la Gestión
Diseñar e implementar una base de datos centralizada que modernice el control de inventario y el flujo de préstamos de la biblioteca.

2. Optimizar las Búsquedas
Desarrollar un módulo de consulta rápida que permita encontrar libros de manera eficiente mediante filtros específicos, tales como autor, fecha de lanzamiento y el grado o año escolar correspondiente.

3. Mejorar la Experiencia de Usuario
Facilitar el acceso a la información tanto para el personal que administra el área como para la comunidad educativa que consulta el catálogo.

Personal (Usuarios del Sistema)
El software está diseñado para responder a las necesidades de dos perfiles clave dentro de la institución:

1. Personal Administrativo
Encargados principales de la gestión y mantenimiento del sistema. Tendrán el control del inventario, registro de nuevos ejemplares, actualización de datos y supervisión global de los préstamos y devoluciones en el área de la biblioteca.

2. Personal Docente
Usuarios que podrán interactuar con el sistema para consultar la disponibilidad del material pedagógico, planificar sus lecturas de acuerdo al grado o año escolar, y gestionar los recursos académicos necesarios para sus planificaciones curriculares.
